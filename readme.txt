Permite mostrar un modal para los terminos y condiciones en cualquier formulario web

== Caracteristicas ==

* Permite mostrar un modal para los terminos y condiciones de un formulario.
* Permite cambiar el color del link y hover del enlace de terminos y condiciones.

**Privacidad**

Este plugin no recopila ni almacena ningún dato de usuario. No establece ninguna cookie ni se conecta a ninguna ubicación de terceros. Por lo tanto, este plugin no afecta la privacidad del usuario de ninguna manera.

== Instalación ==

**Cómo instalar**

* Instala desde el repositorio buscandolo como "Modal Terminos y Condiciones".
* Sube el plugin a tu sitio y actívalo.

[Más información sobre la instalación de plugins WP](https://codex.wordpress.org/Managing_Plugins#Installing_Plugins)

== USO ==

**Cómo usar**

* Configurar el plugin, la pagina de los terminos y condiciones y los colores (ver imagen #1).
* En **Elementor** en el campo tipo "Acceptance" en el campo **Acceptance Text** agregar texto y una etiqueta "a" con la clase "modal_terminos" (ver imagen #2).
* En **Contact Form 7** en el formulario agregar un shortcode [acceptance] y la etiqueta "a" con su clase "modal_terminos" (ver imagen #3).

== Actualización ==

Simplemente haga clic en "Actualizar" en la pagina de plugins y deje que WordPress lo haga por usted automáticamente.
